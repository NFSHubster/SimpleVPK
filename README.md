# SimpleVPK

SimpleVPK is a homebrew app based on [easyVPK](https://github.com/Electric1447/EasyVPK) that's designed for the PlayStationÂ®Vita that allows USERS to easily download & install VPKs & Data Files from [VitaDB](https://www.rinnegatamante.eu/vitadb/#/)


# Changes From easyVPK

- Updated VitaDB's URL to make the app usable again (from rinnegatamante.it/vitadb to rinnegatamante.eu/vitadb)

- Changed LiveArea Assets

# Building

- Prerequisites:
  [VitaSDK](https://vitasdk.org/)

- Compiling:

```sh
mkdir build
cd build

cmake ..
make
```
